## 🚀 **COMO USAR:**

### **1. Teste Local:**
```bash
# Iniciar servidor local
firebase serve --only hosting

# Ou usar o script
./deploy-pwa.sh
```

### **2. Deploy Produção:**
```bash
# Deploy completo
firebase deploy --only hosting

# Ou usar script automatizado
./deploy-pwa.sh
```

### **3. Teste no Celular:**
1. Acesse o site no navegador mobile
2. **Android:** "Adicionar à tela inicial"
3. **iOS:** Compartilhar → "Adicionar à tela inicial"
4. Abra como app nativo!
